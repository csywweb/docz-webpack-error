## 参数

| 参数 | 类型 | 是否必填        |默认值   |说明
|------|------------------|-----------------|---------|----------|
|filter|object|否|{}|接口调用参数，比如deptId
|action|Promise|是||翻页api接口，必须是Promise对象
|columns|array|是|[]|表格列
|total|string|是|0|总的数据条数
|pageSize|number|否|20|分页条数
|list|array|是|[]|表格数据
|...|...|否|undefined|zent的Table组件其他配置都可以使用，参考[Table](https://youzan.github.io/zent/zh/component/table)
